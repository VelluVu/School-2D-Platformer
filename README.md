# School 2D Platformer

> A 2D platformer game built in Unity as a school project — exploring core platformer mechanics.

![Unity](https://img.shields.io/badge/Engine-Unity-black?logo=unity) ![Language](https://img.shields.io/badge/Language-C%23-purple) ![Type](https://img.shields.io/badge/Type-School%20Project-orange)

## About

A 2D platformer game developed in Unity as part of a game programming school curriculum. The project focuses on implementing the foundational mechanics of the platformer genre: player movement, jumping, collision, enemies, and level design.

## Features

- 🎮 Responsive player movement and jumping
- 🏃 Physics-based character controller
- 💀 Enemy AI with basic patrol behaviour
- 🗺️ Level design with platforms, hazards, and collectibles
- ❤️ Player health and death/respawn system
- 🎯 Win condition and basic UI (score/health display)

## Built With

- **Unity** (2D)
- **C#** — gameplay scripting

## Getting Started

### Prerequisites

- Unity Editor (2019.x or later recommended)

### Running the Project

1. Clone the repository:
   ```bash
   git clone https://github.com/VelluVu/School-2D-Platformer.git
   ```
2. Open in Unity Hub and load the project folder.
3. Open the main scene from `Assets/Scenes/`.
4. Press **Play**.

## Context

One of the first complete Unity game projects from game programming studies. Building a platformer from scratch establishes the core skills — physics, input, scene management, and basic AI — that all later game projects build upon.
